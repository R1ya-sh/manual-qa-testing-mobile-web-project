**Mobile Application Testing Project – Duolingo**

## Project Overview
This project demonstrates manual testing of the Duolingo mobile application using structured test cases, execution reports, and bug tracking.

## Tools Used
- Testiny (Test Case Management)
- Excel / Google Sheets
- GitHub (Documentation)

## Testing Scope
- Functional Testing
- UI Testing
- Notification Testing
- UX/UI Suggestions

## Project Files
- Duolingo_TestCases.pdf → Detailed test cases
- Duolingo_TestExecution.xlsx → Execution results
- Duolingo_BugReport.xlsx → Defect tracking
- Duolingo_UX_UI_Suggestions.pdf → Improvement suggestions

## Sample Bug
- In "Listen & Respond" exercise, the application accepts wrong answers and mark them correct.
- Severity: High
- Status: Open
