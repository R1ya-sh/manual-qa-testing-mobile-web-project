# UI/UX Suggestion: Link Visibility Issue

## Description
On the TodoMVC page, the text "TodoMVC" appears as normal text but behaves like a clickable link when hovered or clicked.

## Issue
- The link is not visually distinguishable from regular text
- No underline, color change, or styling indicates it is clickable

## Expected Behavior
Clickable links should be clearly visible to users through standard UI cues such as:
- Underline
- Different color
- Hover effect

## Actual Behavior
The text appears as plain text, which may confuse users and reduce discoverability

## Impact
- Poor user experience
- Users may not realize the element is interactive

## Suggestion
Apply standard link styling (e.g., underline or color differentiation) to improve visibility and usability
